# Hendl Heinrich - Online-Reservierungssystem

## Das Problem heute

### Telefonische Vorbestellung
- Telefon klingelt während der Arbeit am Grill
- Hände sind fettig/beschäftigt - Telefon abnehmen schwierig
- **Lange Schlangen** - Kunden warten, weil Mitarbeiter am Telefon sind
- Kunden müssen wissen, welche Nummer zu welchem Wagen gehört
- Missverständnisse am Telefon möglich ("3 Hendl oder 3 halbe?")

### Zettelwirtschaft
- Reservierungen werden handschriftlich notiert
- Oft unleserlich oder schlampig geschrieben
- Am Ende des Tages mühsam durchzählen
- Kostet Zeit und ist fehleranfällig
- Keine Übersicht wie viele Reservierungen für den Tag vorliegen

---

## Die Lösung: Hendl Heinrich App

### Einfache Online-Reservierung

**Für Kunden:**
- Reservierung bequem von zuhause aus
- Übersicht aller Standorte mit Karte
- Genauer Standort der Wägen sichtbar
- Bestätigung per Code - kein Vergessen mehr
- Reservierung selbst stornieren möglich

**Für die Mitarbeiter:**
- **Der Zettel fällt komplett weg!**
- Bei Telefonanrufen: Reservierung selbst ins System eintragen
- Jederzeit Übersicht: Wie viele Hendl sind aktuell reserviert?
- Reservierungen abhaken wenn abgeholt
- Kein mühsames Durchzählen mehr am Tagesende

---

## Vergleich: Alt vs. Neu

| Heute (Telefon & Zettel) | Neu (App) |
|--------------------------|-----------|
| Telefon klingelt beim Arbeiten | Reservierungen kommen automatisch rein |
| Lange Schlangen durch Ablenkung | Volle Konzentration auf Kunden vor Ort |
| Handschriftliche Zettel, oft unleserlich | Digitale Übersicht, immer lesbar |
| Mühsames Durchzählen am Tagesende | Übersicht auf Knopfdruck |
| "Welche Nummer für welchen Wagen?" | Eine Website für alles |
| Missverständnisse möglich | Alles schriftlich & eindeutig |
| Kunde muss Standort kennen | Karte zeigt genauen Standort |

---

## So funktioniert's

### Für den Kunden (3 Schritte):

1. **Standort wählen** - "Wo will ich abholen?"
2. **Hendl reservieren** - Name, Anzahl, fertig
3. **Code erhalten** - Damit abholen

### Für die Mitarbeiter:

1. Morgens: Übersicht der Reservierungen
2. Bei Abholung: Kunde nennt Code → abhaken
3. Fertig!

## Telefon bleibt, Zettel geht!

### Was sich ändert:

| | Vorher | Nachher |
|---|--------|---------|
| **Zettel** | Handschriftlich, unleserlich | **Fällt komplett weg!** |
| **Telefon** | Bleibt erstmal | Wird über Zeit weniger |

### So funktioniert's:
- Kunde ruft an → Mitarbeiter trägt Reservierung ins System ein
- Kein Zettel mehr, alles digital
- Über Zeit: Immer mehr Kunden nutzen die App selbst
- Telefon wird schrittweise weniger wichtig

**Ergebnis:** Sofort kein Zettel-Chaos mehr!

## Zusammenfassung

| Vorteil | Nutzen |
|---------|--------|
| Weniger Telefonanrufe | Mehr Zeit für die Arbeit am Grill |
| 24/7 Reservierungen | Kunden können auch abends buchen |

**Das Ergebnis:** Weniger Stress, zufriedenere Kunden, moderner Auftritt
